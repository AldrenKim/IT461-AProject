## Scripts

Install needed client and server packages and builds
`npm i`

`npm start`

Run client only
`npm run client`

Run server only
`npm run server`

## Prereqs

If you're getting not allowed errors

Run `npm run mysql` then paste these on the terminal

```
CREATE USER 'root'@'%' IDENTIFIED BY 'root';
grant all on *.* to 'root'@'%';
```
